Módulo I

Linguagem de Programação Python.

Histórico

Guido Van Rossum foi o criador e responsável pela existência do Python. Trata-se de um informático de origem holandesa que foi responsável por projetar, pensar e definir todas as formas possíveis de evolução desta popular linguagem de programação.

Objetivos

O principal objetivo da linguagem Python é estimular produtividade e legibilidade de códigos. Ela é conhecida por sua velocidade e pelo seu estilo compacto de escrever. Com poucas linhas de código é possível criar muitas funcionalidades.

Características

Python é uma linguagem interpretada e pode ser facilmente executada em linha de comando. Existem cursos de aprendizado para programar em Python que ajudam a dar o pontapé inicial. A quantidade de código usada em uma função em Python pode ser entre 3 e 5 vezes menor do que os códigos feitos em Java. Esse número sobe para entre 5 e 10 vezes quando se trata de C++. A sintaxe de programação é muito intuitiva e causa menos confusão ao programador. Por exemplo, não é necessário ficar abrindo e fechando chave toda vez que uma nova função for definida. O Python possui regras simples que facilitam a vida de quem escreve o código. A linguagem Python possui uma comunidade muito ativa que produz e traduz muito conteúdo para outros usuários. Existem muitos l ivros, tutoriais e manuais de como usar a maioria das bibliotecas criadas para a linguagem. A maioria das bibliotecas disponibilizadas são open source. Isso permite uma inserção maior dos programadores em uma área de aplicação específica como desenvolvimento web ou análise de dados, por exemplo.

Ambiente, Recursos básicos de maquina, instalação e configuração

Dá suporte ao Big Data (visualização de dados), termo para descrever a quantidade imensa de dados que são produzidos atualmente e disseminados com o apoio da internet. Eles são gerados em alta velocidade, em uma grande variedade de formatos e de forma estruturada ou não, ou seja, são dados mais complexos e requerem tecnologias específicas. O R se destaca como uma das principais soluções a fim de lidar com esse desafio.
